# IntelliHire – AI-Powered Resume Parser & Candidate Screening System

IntelliHire is an AI-driven recruitment automation platform that streamlines the resume screening process by extracting, analyzing, and organizing candidate information from resumes. Leveraging Natural Language Processing (NLP) and intelligent matching techniques, the system converts unstructured resume data into structured candidate profiles, enabling faster and more accurate hiring decisions.

The platform automatically identifies key information such as skills, education, work experience, certifications, and projects from PDF and DOC resumes. It further evaluates candidate suitability through resume-to-job matching and ranking mechanisms, helping recruiters efficiently identify the most relevant applicants.

### Key Highlights

* AI-powered resume parsing and information extraction
* NLP-based skill, education, and experience identification
* Automated candidate profile generation
* Resume-to-job description matching
* Candidate ranking and screening
* Structured data storage using MySQL
* Scalable and recruiter-friendly architecture

### Tech Stack

**Python • NLP • Machine Learning • Regex • MySQL • REST APIs • Git • GitHub**

### Objective

To reduce manual effort in recruitment by automating resume analysis and candidate screening while demonstrating the practical application of Artificial Intelligence and Natural Language Processing in real-world hiring systems.
